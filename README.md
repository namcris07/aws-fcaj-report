# AWS FCAJ Internship Report 🌟

Chào mừng bạn đến với báo cáo thực tập của nhóm mình tại chương trình **AWS First Cloud AI Journey (FCAJ)**! 👋

## 📋 Giới thiệu

Xin chào! Chúng mình là nhóm thực tập sinh dự án **DevSecOps Factory trên AWS**, hiện đang tham gia chương trình **AWS First Cloud AI Journey (FCAJ)**. Website này là nơi nhóm mình ghi lại toàn bộ hành trình thực tập 9 tuần đầy thú vị với dự án *"Xây dựng hệ thống CI/CD DevSecOps end-to-end cho ứng dụng Web trên Amazon EKS"*.

Dự án tự động hóa quy trình kiểm thử bảo mật (**Shift-Left Security**) tích hợp 6 cổng an ninh (Gitleaks, Trivy, SonarQube, Checkov, OWASP ZAP) và triển khai GitOps tự động với Argo CD trên hạ tầng Amazon EKS.

**Thời gian thực tập:** Từ 15/06/2026 đến 14/08/2026 (9 tuần)

- **AWS Cloud Services**: Amazon EKS, Amazon ECR, Amazon CloudWatch Container Insights, AWS IAM (IRSA), AWS Load Balancer Controller
- **DevOps & GitOps**: Jenkins (CI Engine), Argo CD (GitOps Engine), Docker, Kustomize, Helm, k3d (Local Kubernetes)
- **Security Gates**: Gitleaks (Secrets), Trivy (SCA & Container Scan), SonarQube (SAST), Checkov (IaC Scan), OWASP ZAP (DAST)
- **Website Engine**: Hugo Static Site Generator (`hugo-theme-learn`)
- **Hosting & Source Control**: GitHub Pages & GitHub Repositories
- **Languages**: English & Vietnamese (Song ngữ Việt - Anh)
